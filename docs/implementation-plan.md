# Implementation Plan

## Phase 1: Production Foundation

- Set up Next.js, TypeScript, Prisma, PostgreSQL.
- Build public page shell.
- Build admin shell with role-aware navigation.
- Create database schema and seed data.
- Add authentication.

## Phase 2: Donation Engine

- Implement donation checkout.
- Add M-Pesa STK Push.
- Add payment callback handling.
- Add donation records.
- Generate receipts.
- Add finance reconciliation queue.

## Phase 3: Campaigns And Sponsorship

- Campaign CRUD.
- Campaign landing pages.
- Beneficiary profiles.
- Consent and medical review workflow.
- Child sponsorship donations.

## Phase 4: Events

- Event CRUD.
- Ticket products.
- Registration checkout.
- QR tickets.
- Check-in dashboard.

## Phase 5: CRM And Marketing

- Donor profiles.
- Segments.
- Communication preferences.
- Email/SMS journeys.
- Lapsed donor and failed payment workflows.

## Phase 6: Corporate And Reporting

- Corporate partner pipeline.
- Sponsorship packages.
- Partner reports.
- Public impact dashboard.
- Board-ready reporting exports.
